# Advanced C Programming Practice Collection

A curated collection of advanced-level C programming exercises designed to strengthen understanding of data structures, pointers, file handling, and algorithmic problem-solving.

This repository builds upon fundamental and intermediate C programming concepts and introduces more practical programming techniques commonly used in real-world software development.

The goal of this collection is to improve logical thinking, algorithm design, and program structuring using the C programming language.

---

## 📂 Programs Included

### 1️⃣ Binary Search

Implements the binary search algorithm to efficiently find an element in a sorted array.

### 2️⃣ Bubble Sort

Sorts an array of numbers using the bubble sort algorithm.

### 3️⃣ Selection Sort

Sorts an array by repeatedly selecting the smallest element.

### 4️⃣ Insertion Sort

Builds a sorted array one element at a time using insertion sort.

### 5️⃣ Matrix Addition

Adds two matrices of the same dimension.

### 6️⃣ Matrix Multiplication

Performs multiplication of two matrices.

### 7️⃣ Transpose of Matrix

Calculates the transpose of a given matrix.

### 8️⃣ Swap Using Pointers

Swaps two numbers using pointer variables.

### 9️⃣ String Reverse Using Pointers

Reverses a string using pointer manipulation.

### 🔟 Student Record Using Structures

Stores and displays student data using structures.

### 1️⃣1️⃣ Employee Salary Structure

Manages employee records and calculates salary.

### 1️⃣2️⃣ File Write Program

Writes user input data to a file using file handling.

### 1️⃣3️⃣ File Read Program

Reads and displays contents from a file.

### 1️⃣4️⃣ File Copy Program

Copies content from one file to another.

### 1️⃣5️⃣ Dynamic Memory Allocation

Demonstrates memory allocation using `malloc()` and `free()`.

### 1️⃣6️⃣ Linked List Creation

Creates a singly linked list dynamically.

### 1️⃣7️⃣ Linked List Traversal

Traverses and prints elements of a linked list.

### 1️⃣8️⃣ Stack Using Array

Implements stack operations (push and pop) using arrays.

### 1️⃣9️⃣ Queue Using Array

Implements queue operations using arrays.

### 2️⃣0️⃣ Linear Search

Searches for an element in an array using linear search.
